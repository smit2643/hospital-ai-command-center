# Product Roadmap

## Phase 1 (Current)
- Core RBAC workflows
- OCR pipeline for image lab reports
- Tokenized e-sign flow and signed artifact storage
- Async architecture and Docker deployment

## Phase 2 (Near-Term)
- PDF OCR pipeline with page splitting
- Better extraction models (hybrid rules + ML)
- Notification center and in-app task inbox
- Fine-grained role permissions and delegated admins

## Phase 3 (Scale)
- Multi-hospital tenancy
- EHR/HL7/FHIR connectors
- Analytics dashboards (turnaround time, OCR accuracy, signature SLA)
- Compliance packs (policy templates, retention automation)

## Phase 4 (Advanced AI)
- Smart triage and anomaly detection on lab trends
- Doctor-assistant copilots for report summarization
- Clinical workflow recommendations
